## [0.1.0] - 2026-04-18

### Added

- Initial release
- Core package with retrieval, AI calling, and JSON parsing
- Server package with Express middleware
- React package with AiAssistant modal and AiButton components
- Express + MongoDB example
- Complete documentation

### Features

- 🚀 Plug-and-play integration
- 🎯 Smart retrieval with keyword matching
- 🔗 Citation with source tracking
- 📦 Lightweight (no vector database required)
- 🎨 Theme customization
- 🌐 Dual mode: article-bound and general Q&A
- 🔄 Category expansion for better results
- 🛡️ Graceful degradation
